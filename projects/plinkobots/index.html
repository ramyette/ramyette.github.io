PlinkoBots

PlinkoBots is a Godot game programmed with GDScript that is similar to the game Peggle. It consists of a series of levels in which the player must shoot a ball with a launcher in order to clear all pegs that contain robots. The graphics and controls are simple, and are explained within the game.

The game is downloadable and playable on both Windows and Mac. Click the Itch.io button below for more info!

This is my first ever game project, and is something that took me quite a while to develop. However throughout the development, I learned a lot about graphics, game development, and software engineering in general.
